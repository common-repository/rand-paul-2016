== Description ==

When activated, this plugin will place an image with link on the top right corner of your Wordpress powered website.

== Installation ==

Visit the "Ron Paul 2012" options page, choose your settings, save changes, done.

Update (2012/11/08): Well the elections are over and Ron Paul was not elected to be the next president of the United States. Deeper implications aside, the Ron Paul 2012 plugin is now discontinued. If you&rsquo;re looking for a WordPress plugin to display a banner, button, or badge on your website, check out my new plugin [Show Support Ribbon](http://perishablepress.com/show-support-ribbon/).
